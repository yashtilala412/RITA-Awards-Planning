# RITA-Awards-Planning
This is web appication in which RITA(RITA, a non-profit IT industry association, is the apex body for all IT Industires specifically located in Rajkot City and Saurashtra region (Gujarat) India.) Wants to give awards to
Best IT company in different domain(like best cloud ,best cyber security,best game developer,and 25 more..),so this application helps them to collect nomination form data.
Technologies:HTML,CSS,JS,tooljet,SMTP,AWS S3
Database:Postgresql
